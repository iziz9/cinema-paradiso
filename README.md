# CINEMA-PARADISO
TMDB API를 이용한 영화 검색 사이트입니다.

## 배포 링크

[✨ 배포 링크](https://cinema-paradiso-green.vercel.app/)

## 설치 및 실행방법

```shell
$ git clone https://github.com/iziz9/cinema-paradiso

$ cd cinema-paradiso

$ yarn install

$ yarn start
```
